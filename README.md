# CLI Vector Editor

Простой векторный редактор с интерфейсом командной строки (CLI), написанный на Python.

## Описание
Программа позволяет создавать, просматривать и удалять базовые геометрические фигуры в виртуальном холсте (в памяти оболочки):
- Точка (Point)
- Отрезок (Line)
- Круг (Circle)
- Квадрат (Square)

## Системные требования
- Python 3.7+

## Запуск
1. Клонируйте репозиторий.
2. В терминале перейдите в папку проекта.
3. Выполните команду:
   ```bash
   python main.py
   ```

## Доступные команды
- `add_point <x> <y>` — добавить точку
- `add_line <x1> <y1> <x2> <y2>` — добавить отрезок
- `add_circle <x> <y> <radius>` — добавить круг (radius > 0)
- `add_square <x> <y> <side>` — добавить квадрат (side > 0)
- `list` — вывести список всех созданных фигур с их ID
- `remove <id>` — удалить фигуру по её ID
- `help` — показать справку по всем командам
- `exit` (или `quit`) — выйти из редактора
